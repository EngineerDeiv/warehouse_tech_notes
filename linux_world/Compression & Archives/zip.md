
## ZIP Archives

### Create

Create ZIP archive.

```bash
zip file.zip files
```

Create ZIP archive recursively.

```bash
zip -r file.zip folder/
```

Create ZIP archive with maximum compression.

```bash
zip -9 -r file.zip folder/
```

### Add Files

Add files to an existing ZIP archive.

```bash
zip file.zip newfile
```

### Extract

Extract ZIP archive.

```bash
unzip file.zip
```

Extract into a specific directory.

```bash
unzip file.zip -d /path/to/dir
```

### List Contents

List archive contents.

```bash
unzip -l file.zip
```
