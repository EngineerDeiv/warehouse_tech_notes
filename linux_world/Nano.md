# Nano Basics

Open or create a file:

```bash
nano file
```

Save file:

- `Ctrl + O` (Write Out), then Enter to confirm filename

Exit:

- `Ctrl + X`

Cut (remove) current line:

- `Ctrl + K`

Uncut (paste):

- `Ctrl + U`

Search within file:

- `Ctrl + W`, type search term, Enter

Replace (search and replace):

- `Ctrl + \` then follow prompts

Help screen:

- `Ctrl + G`
