# Compression

Create tar archive.

```bash
tar cf file.tar files
```

Extract tar archive.

```bash
tar xf file.tar
```

Create compressed tar archive.

```bash
tar czf file.tar.gz
```

Extract compressed tar archive.

```bash
tar xzf file.tar.gz
```

Create ZIP archive.

```bash
zip -r file.zip files
```

Extract ZIP archive.

```bash
unzip file.zip
```
