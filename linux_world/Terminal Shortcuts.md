# Terminal Shortcuts

Stop current command.

```text
Ctrl + C
```

Suspend current command.

```text
Ctrl + Z
```

Resume in background.

```bash
bg
```

Resume in foreground.

```bash
fg
```

Repeat last command.

```bash
!!
```

Run command number n.

```bash
!n
```

Show command history.

```bash
history
```
