# Process Management

```bash
ps
```

Display active processes.

```bash
top
```

Display running processes.

```bash
kill pid
```

Terminate a process.

```bash
killall proc
```

Terminate all matching processes.

```bash
bg
```

Resume stopped job in background.

```bash
fg
```

Bring job to foreground.

```bash
fg n
```

Bring job number n to foreground.
