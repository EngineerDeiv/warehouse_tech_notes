# System Information

Show current date and time.

```bash
date
```

Show system uptime.

```bash
uptime
```

Current user.

```bash
whoami
```

Current user ID information.

```bash
id
```

Kernel and system information.

```bash
uname -a
```

CPU information.

```bash
cat /proc/cpuinfo
```

Memory information.

```bash
cat /proc/meminfo
```

Disk usage.

```bash
df
```

Directory space usage.

```bash
du
```

Memory and swap usage.

```bash
free
```

Possible application locations.

```bash
whereis app
```

Executable used by default.

```bash
which app
```
