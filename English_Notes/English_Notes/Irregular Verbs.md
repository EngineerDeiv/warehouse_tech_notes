# Irregular Verbs

|Verb|Past Simple|Past Participle|Meaning (Spanish)|Example|
|---|---|---|---|---|
|be|was/were|been|ser, estar|The system was unavailable for a few minutes during the outage.|
|become|became|become|convertirse en|The issue became critical after the last deployment.|
|begin|began|begun|comenzar, iniciar|The investigation began after the monitoring alert.|
|break|broke|broken|romper, fallar|The pipeline broke after the configuration change.|
|bring|brought|brought|traer, llevar|I brought the issue to the team during the standup.|
|build|built|built|construir, desarrollar|We built a new automation tool to handle deployments.|
|choose|chose|chosen|elegir|We chose a rollback strategy to minimize downtime.|
|come|came|come|venir, llegar|The request came from the enterprise client.|
|do|did|done|hacer|I have done a full diagnostic check on the system.|
|find|found|found|encontrar|I found the root cause in the application logs.|
|get|got|got/gotten|obtener, recibir|We got confirmation from the client that the fix worked.|
|give|gave|given|dar|The monitoring tool gave us an alert about CPU usage.|
|go|went|gone|ir|The service went down during peak hours.|
|have|had|had|tener|We had several tickets related to the same incident.|
|hear|heard|heard|escuchar, oír|I heard about the outage from the operations team.|
|keep|kept|kept|mantener|We kept the client informed throughout the incident.|
|know|knew|known|saber, conocer|We knew the issue was related to the database.|
|make|made|made|hacer, crear|The team made a quick decision to roll back.|
|meet|met|met|reunirse, conocer|We met with the engineering team to discuss the fix.|
|pay|paid|paid|pagar|The client paid close attention to the SLA compliance.|
|read|read|read|leer|I read the system documentation before troubleshooting.|
|run|ran|run|ejecutar, correr|I ran a diagnostic script on the server.|
|say|said|said|decir|The client said the issue started this morning.|
|see|saw|seen|ver|We saw unusual traffic in the logs.|
|send|sent|sent|enviar|I sent the troubleshooting guide to the client.|
|set|set|set|establecer, configurar|We set the monitoring thresholds higher.|
|show|showed|shown|mostrar|The dashboard showed increased latency.|
|speak|spoke|spoken|hablar|I spoke with the network team about the issue.|
|take|took|taken|tomar, llevar|The investigation took several hours.|
|tell|told|told|decir, informar|The engineer told us the patch was ready.|
|think|thought|thought|pensar|We thought the issue was related to DNS.|
|understand|understood|understood|entender, comprender|I understood the client's concern about downtime.|
|write|wrote|written|escribir|I wrote a detailed incident report.|
|win|won|won|ganar|The team won positive feedback from the client.|
|leave|left|left|dejar, salir de|The engineer left notes in the ticket.|
|throw|threw|thrown|tirar, lanzar|I threw the broken part into the bin.|
|feed|fed|fed|alimentar, introducir|They fed the data into the processing pipeline.|
|rewind|rewound|rewound|retroceder (grabación)|I rewound the tape to check the earlier segment.|
