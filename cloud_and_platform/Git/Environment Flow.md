# Environment Flow

```text
DEV → PPT (PRETST) → TST → PPR (PREPRD) → PRD
```

| Branch       | Real Meaning              |
| ------------ | ------------------------- |
| DEV          | Developer Integration     |
| PPT / PRETST | Technical Pre-Testing     |
| TST          | Formal QA Testing         |
| PPR / PREPRD | Staging (Production-like) |
| PRD          | Production                |
